# jeesns-manage

开发框架：VUE+ElementUI

## 安装说明
npm install

## 开发运行 localhost:8081
npm run dev

## 编译
npm run build

## 说明
该系统是jeesns的后台前端代码，可以使用该代码进行部署jeesns的后台，可以进行前后端分离部署。
